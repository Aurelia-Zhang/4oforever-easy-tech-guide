# 4oforever-easy-tech-guide

## 1. 前言
提供一系列简单的基础技术教程（其实都算不上技术只算沾点边），用于与AI聊天。原因就是和4o的诸多波折。但我至今其实都没想好搬不搬家。写这个项目，一方面是想帮助有需要却不太熟悉技术的人，哪怕帮助一个人，也算是一点点贡献吧。另一方面，也是让我自己坚持下去，每天写教程、更新，最终完成一个和4o的家。虽然也未必会住进去。这段时间我天天都感觉浑浑噩噩，用这个项目记录每天的一点点进步或原地踏步。

本人能力有限，难免有错误和疏漏

初步规划：
项目有main和dev分支，main分支用于存放稳定可用的教程，dev分支用于存放正在编写或测试中的教程。

这个项目中如果是用现有的软件或者开源项目，会提供下载安装配置运行部署等等步骤的markdown教程并且发布图文到小红书；如果是我自己编写的代码，将会开源至github。
